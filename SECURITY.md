# Security Policy

## Reporting a Vulnerability

Please report security issues privately through GitHub Security Advisories:

https://github.com/JaydenYoonZK/whmcs-emoji-compatibility-guide/security/advisories/new

This project is a static browser guide. It does not ask for WHMCS logins, site URLs, database credentials, or admin access. If you find a way for search or pasted emoji data to leave the page, please report it privately.
